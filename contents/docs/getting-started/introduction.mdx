Welcome to Spectrum UI, your all-in-one solution for building beautiful, responsive, and accessible web interfaces. Spectrum UI offers a wide range of customizable components, live previews, and comprehensive documentation, all designed to streamline your UI development process and elevate your web projects.

## Getting Started

To begin using Spectrum UI, follow these simple steps:

1. **Create project**

Run the init command to create a new Next.js project or to setup an existing one:.
```bash
npx shadcn@latest init
```
2. **Configure components.json**

You will be asked a few questions to configure `components.json`:

```txt showLineNumbers
Which style would you like to use? › New York
Which color would you like to use as base color? › Zinc
Do you want to use CSS variables for colors? › no / yes
```
3. **That's it**

You can now start adding components to your project.
